### Deploy
Auto deploy from Travis CI to Heroku  
[link to app](https://bmstu-graphics.herokuapp.com)

### Run
#### Install dependencies
```bash
    npm i
```

#### Start webpack
```bash
    npm run webpack
```

#### Start webpack hot server
```bash
    npm start
```

#### Start express server
```bash
    npm run server
```
