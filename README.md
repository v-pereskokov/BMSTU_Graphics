## BMSTU computer graphics course

### Seminars
```bash
    git checkout seminar-[seminar number]
```  
Example:  
```bash
    git checkout seminar-1
```

### Run
#### Install dependencies
```bash
    npm i
```

#### Start webpack
```bash
    npm run webpack
```

#### Start webpack hot server
```bash
    npm start
```

#### Start express server
```bash
    npm run server
```
